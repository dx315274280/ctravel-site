**WeChat Pay** is how China actually pays — metro, taxis, street food, many hotels. Tourists can link select **international credit/debit cards** without a Chinese bank account.

## Step-by-step setup

1. Install **WeChat** and register with your mobile number
2. Go to **Me → Services → Wallet**
3. Add **bank card** (Visa/Mastercard — availability varies by country)
4. Complete identity verification if prompted
5. Enable **Pay / Receive** QR codes

## Where it works

- Most chain restaurants and supermarkets
- Shenzhen metro (via transit QR inside WeChat/Alipay)
- DiDi rides
- Many tourist sites

Cash is backup, not primary, in tier-1 cities.

## Alipay alternative

**Alipay Tour Pass** (when available) is equally important — install both apps.

## Common issues

| Problem | Fix |
|---------|-----|
| Card rejected | Try another card; some prepaid cards fail |
| Verification loop | Use hotel Wi-Fi; ensure passport name matches |
| Merchant only Alipay | Ask or pay cash |

## Safety

Use official app stores only. Never share SMS codes. Set a payment PIN.
